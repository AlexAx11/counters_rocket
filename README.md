# counters_rocket
The counters count down the time until the rocket takes off. 
Each counter is reset by its own button. At the same time, the remaining counters retain their values.

1. Create a new React typescript project:
npx create-react-app my-app --template typescript
or
yarn create react-app my-app --template typescript

2. Add all files into folder 'src'. Replace files 'App', 'index' in the folder 'src'.

3. Start the project.
